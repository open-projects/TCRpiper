TCRpiper

This is a test version of TCRpiper, a web-based application for analysis of T-cell receptor repertoires. It doesn't have too much fancy flare to it, but it makes a great use of the standard programs for TCR sequencing data generated on Illumina instruments. Feel free to use this application for your project!
